# Flutter_Service_App
An app that provides users with local businesses in their area and allows then to schedule services from them.


# Learning Opportunities
- Api authentication
- Sql database usage
- Good Ui/Ux design

# Goals
- Rest Api integration
- Uses a real encrypted sql database
- Monetization through ads
- Subscription service for the business 
- 2 apps - Client and Owner-side
- 1000 users
